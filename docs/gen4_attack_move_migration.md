# 4世代攻撃技対応メモ

対象バージョンは DPPt/HGSS/PBR 相当です。状態技は今回の追加対象外で、ダメージが発生する4世代追加技だけを `pokemon_moves.json` に追加します。

## 現在の対応状況

| 項目 | 状態 |
| --- | --- |
| 4世代追加攻撃技 | 80件追加済み |
| なみのり対象 | `target: 3` に変更済み |
| 範囲技補正 | `target: 2` と `target: 3` をダブル時0.75倍対象に変更済み |
| めざめるパワー | 4世代通り常に特殊に変更済み |
| ウェザーボール | 天候でタイプ変更、分類は常に特殊、天候時は威力2倍に変更済み |
| 2倍条件技 | `two_fold` を威力2倍として処理するよう変更済み |
| 体重依存技 | 既存の `weight_based` で `くさむすび` に対応済み |
| ジャイロボール | 攻撃側/防御側の素早さ実数値から威力計算済み |
| しぼりとる/にぎりつぶす | 防御側の現在HP/最大HPから威力計算済み |
| 既存3世代技の物理/特殊分類 | 4世代の技ごとの分類へ更新済み |
| 既存技の4世代威力変更 | ダメージに関係する10件を更新済み |
| 4世代ポケモンデータ | 387-493と主要フォルムを追加済み |
| ポケモン別候補技 | DPPt/HGSSの攻撃技候補を反映済み |
| 4世代ダメージ関連特性 | 主要な攻撃/防御補正を追加済み |

## target の扱い

| target | 意味 | 範囲補正 |
| --- | --- | --- |
| `1` | 通常単体 | なし |
| `2` | 相手全体 | ダブル時0.75倍 |
| `3` | 自分以外3体 | ダブル時0.75倍 |
| `0` | 自分またはダメージ計算対象外 | なし |

4世代の `なみのり`、`ほうでん`、`ふんえん` は `target: 3` とします。

## 威力補正 class

| class | 対応技 | 内容 |
| --- | --- | --- |
| `two_fold` | めざましビンタ、しおみず、しっぺがえし、ダメおし、ゆきなだれ等 | `2倍条件を満たしたとき` チェックで威力2倍 |
| `weather_ball` | ウェザーボール | 天候時に威力2倍、タイプ変更、分類は特殊固定 |
| `weight_based` | くさむすび | 防御側の体重で20/40/60/80/100/120 |
| `gyro_ball` | ジャイロボール | `min(150, floor(25 * 相手S / 自分S) + 1)` |
| `target_hp_rate` | しぼりとる、にぎりつぶす | `floor(120 * 相手現在HP / 相手最大HP) + 1` |
| `fling` | なげつける | `item.json` に `fling_power` があれば使用、未設定なら0 |
| `natural_gift` | しぜんのめぐみ | 攻撃側の持ち物に `natural_gift_type` / `natural_gift_power` があれば使用、未設定なら威力0 |
| `judgment` | さばきのつぶて | 攻撃側の持ち物に `judgment_type` があればそのタイプへ変更 |

## 4世代ダメージ関連特性

| 特性 | 対応内容 |
| --- | --- |
| てきおうりょく | STABを2倍に変更 |
| すてみ | 反動/外し反動系の対象技を1.2倍 |
| テクニシャン | 威力60以下を1.5倍 |
| てつのこぶし | パンチ系対象技を1.2倍 |
| スナイパー | 急所倍率を3倍に変更 |
| サンパワー | 晴れ時に特殊攻撃1.5倍 |
| ノーマルスキン | 技タイプをノーマルとして計算 |
| いろめがね | いまひとつ時の最終ダメージ2倍 |
| フィルター/ハードロック | 効果抜群時の最終ダメージ0.75倍 |
| たいねつ | ほのお技の威力半減 |
| かんそうはだ | みず技0ダメージ、ほのお技1.25倍 |
| スロースタート | チェック時に物理攻撃半減 |

`ダウンロード` は能力ランク入力で代替するため、個別UI/自動判定は追加していません。

## まだ別作業として残すもの

| 項目 | 理由 |
| --- | --- |
| 既存3世代技の物理特殊分類修正 | 今回の依頼範囲外。次の作業で `pokemon_moves.json` の既存230件を更新する |
| 3世代から4世代で威力が変わった既存技 | 今回の依頼範囲外。例: あなをほる、そらをとぶ、げきりん、リーフブレード等 |
| `きりふだ` | 残りPP入力UIが必要 |
| `おしおき` | 相手の全能力ランク入力UIがないため対応しない。`index.html` の未対応技へ記載済み |
| 未対応技リストの旧項目 | アイスボール等、もともとの未対応技は引き続き未対応 |

既存技の4世代威力変更は `あなをほる=80`、`ダイビング=80`、`そらをとぶ=90`、`とびひざげり=100`、`とびげり=85`、`リーフブレード=90`、`げきりん=120`、`はなびらのまい=90`、`いわくだき=40`、`でんじほう=120` を反映済みです。
